## Notes

- **Row count:** the reference small file currently has **38** rows (not 40); human columns are often only filled for **17** rows per annotator in the uploaded sheets‚ÄĒdescriptives use ‚Äúfilled‚ÄĚ counts.
- **őļ vs LLM for register:** if `register_markedness_llm` is **constant** (e.g. all `1`) on the merged rows, **Cohen‚Äôs őļ is not informative** and is reported as empty (`NaN`); use **accuracy** instead (`accuracy_register_vs_llm`).
- **Numidia:** pairwise **n** can be **9** on some pairs if one item is missing human labels for that rater in the first-10 set.
- **Essay topic:** categorical prompt label from the corpus; summaries use the **mode** (most common), not a numeric ‚Äúaverage.‚ÄĚ
